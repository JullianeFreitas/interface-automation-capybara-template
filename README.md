# project-base-capybara

Projeto base para automação de telas usando Ruby, Capybara e Cucumber.

Antes de qualquer execução, existem alguns requisitos que devem ser obedecidos:

1 - As informações no arquivo de configuração (/system/config/settings.yml) devem estar preenchidas.

2 - Você deve editar a variável de sistema "Path" e adicionar o diretório onde estão os drivers do Chrome e Firefox.
